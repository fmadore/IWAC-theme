# Design Principles

## Aesthetic Philosophy

Modern, professional, academic - but not flashy or gimmicky.

### Core Principles

1. **Subtle Elegance** - Sophisticated without being showy
2. **Academic Appropriateness** - Clean typography for scholarly content
3. **Cultural Sensitivity** - Respectful of Islamic heritage materials
4. **Functional Beauty** - Design serves usability

## Visual Guidelines

### Colors

- Use the primary orange (IWAC brand) as accent, not dominant
- Subtle, muted backgrounds
- High contrast for readability
- Consistent use of design tokens

### Typography

- Clean, readable fonts
- Generous line height (1.5-1.7)
- Proper heading hierarchy
- Comfortable reading width (65-75 characters)

### Spacing

- Consistent spacing using design tokens
- Generous whitespace
- Clear visual hierarchy
- Breathing room between sections

### Shadows & Depth

- Subtle shadows for elevation
- Avoid heavy drop shadows
- Use glow effects sparingly on interactive elements

### Animations

- Subtle, purposeful motion
- Fast transitions (150-200ms)
- No bouncing/elastic effects
- Respect prefers-reduced-motion

## What to Avoid

- Overly flashy animations
- Jarring color combinations
- Too many competing visual elements
- Unnecessary decorative elements
- Web 2.0 glossy effects
- Harsh borders

## Interactive States

- Clear hover feedback
- Visible focus states
- Subtle scale/lift on hover
- Primary-colored glow for emphasis
