# Design Tokens Quick Reference

## Spacing

| Token | Value | Use Case |
|-------|-------|----------|
| `--space-1` | 0.25rem (4px) | Tiny gaps |
| `--space-2` | 0.5rem (8px) | Small gaps |
| `--space-3` | 0.75rem (12px) | Compact padding |
| `--space-4` | 1rem (16px) | Standard gap |
| `--space-6` | 1.5rem (24px) | Section spacing |
| `--space-8` | 2rem (32px) | Large sections |
| `--space-12` | 3rem (48px) | Major sections |

## Typography

| Token | Size | Use Case |
|-------|------|----------|
| `--text-xs` | ~12-13px | Captions, labels |
| `--text-sm` | ~14-15px | Secondary text |
| `--text-base` | ~16-17px | Body text |
| `--text-lg` | ~18-20px | Lead paragraphs |
| `--text-xl` | ~22-26px | H4 |
| `--text-2xl` | ~28-36px | H3 |
| `--text-3xl` | ~36-48px | H2 |
| `--text-4xl` | ~44-60px | H1 |

## Shadows

| Token | Depth | Use Case |
|-------|-------|----------|
| `--shadow-xs` | Subtle | Borders |
| `--shadow-sm` | Light | Cards at rest |
| `--shadow-md` | Medium | Raised cards |
| `--shadow-lg` | Deep | Modals, dropdowns |
| `--shadow-xl` | Dramatic | Hero elements |

## Glows (Primary-colored shadows)

| Token | Use Case |
|-------|----------|
| `--glow-xs` | Subtle button glow |
| `--glow-sm` | Button hover |
| `--glow-md` | Featured elements |
| `--glow-lg` | Hero buttons |

## Easing & Transitions

| Token | Duration | Use Case |
|-------|----------|----------|
| `--transition-fast` | 150ms | Micro-interactions |
| `--transition-base` | 200ms | Standard transitions |
| `--transition-slow` | 300ms | Page transitions |
| `--ease-out-quart` | - | Snappy UI motion |
| `--ease-out-expo` | - | Dramatic reveals |

## Lifts (Hover elevation)

| Token | Value | Use Case |
|-------|-------|----------|
| `--lift-xxs` | -1px | Buttons, small elements |
| `--lift-xs` | -2px | Cards, content blocks |
| `--lift-sm` | -4px | Entrance animations only |

## Border Radius

| Token | Value | Use Case |
|-------|-------|----------|
| `--radius-sm` | 6px | Buttons, inputs |
| `--radius-md` | 12px | Cards |
| `--radius-lg` | 16px | Large cards |
| `--radius-xl` | 24px | Hero sections |
| `--radius-full` | 9999px | Pills, avatars |

## Colors (Semantic)

| Token | Purpose |
|-------|---------|
| `--ink` | Primary text |
| `--ink-light` | Secondary text |
| `--muted` | Disabled/placeholder |
| `--surface` | Background |
| `--surface-raised` | Elevated background |
| `--border` | Standard borders |
| `--primary` | Brand/accent color |
